# LoongArch 单周期 CPU

本项目是一个基于 LoongArch 指令集的单周期 CPU RTL 实现，使用 Verilog 编写，可在 Xilinx Vivado 中仿真与综合。

## 工程结构

```
loongarch_cpu.srcs/
├── sources_1/new/
│   ├── mycpu_top.v   — CPU 顶层模块（取指、译码、执行、访存、写回）
│   ├── alu.v         — 算术逻辑单元
│   ├── pc_reg.v      — PC 寄存器（复位地址 0x1C000000）
│   ├── regfile.v     — 32 × 32 位通用寄存器堆（r0 硬连线为 0）
│   ├── inst_ram.v    — 指令存储器（1 KB，只读）
│   └── data_ram.v    — 数据存储器（4 KB，支持字节写使能）
└── sim_1/new/
    ├── tb_mycpu.v    — CPU 整体仿真测试台
    ├── tb_alu.v      — ALU 单元测试台
    ├── tb_regfile.v  — 寄存器堆测试台
    └── tb_pc_reg.v   — PC 寄存器测试台
```

## 已实现指令（共 35 条）

> **计数说明**：共实现 **35 条**独立指令编码（不含 `NOP` 别名）。`NOP` 是 `ANDI r0, r0, 0` 的别名，若将其计为独立助记符则为 **36 条**。
> 分类合计：三寄存器型 11 + 移位立即数 3 + 12 位立即数 6 + 20 位立即数 2 + 跳转 5 + 访存 8 = **35 条**。

### 一、三寄存器型指令（R 型，11 条，格式：`op rd, rj, rk`）

| 指令      | 编码（[31:15]）       | 功能描述                                      |
|-----------|----------------------|-----------------------------------------------|
| `ADD.W`   | `17'b00000000000100000` | `rd = rj + rk`，32 位有符号加法              |
| `SUB.W`   | `17'b00000000000100010` | `rd = rj - rk`，32 位有符号减法              |
| `SLT`     | `17'b00000000000100100` | `rd = (rj < rk) ? 1 : 0`，有符号比较        |
| `SLTU`    | `17'b00000000000100101` | `rd = (rj < rk) ? 1 : 0`，无符号比较        |
| `NOR`     | `17'b00000000000101000` | `rd = ~(rj \| rk)`，按位或非                 |
| `AND`     | `17'b00000000000101001` | `rd = rj & rk`，按位与                        |
| `OR`      | `17'b00000000000101010` | `rd = rj \| rk`，按位或                      |
| `XOR`     | `17'b00000000000101011` | `rd = rj ^ rk`，按位异或                     |
| `SLL.W`   | `17'b00000000000101110` | `rd = rj << rk[4:0]`，逻辑左移               |
| `SRL.W`   | `17'b00000000000101111` | `rd = rj >> rk[4:0]`，逻辑右移（补 0）       |
| `SRA.W`   | `17'b00000000000110000` | `rd = rj >>> rk[4:0]`，算术右移（补符号位）  |

### 二、移位立即数指令（3 条，格式：`op rd, rj, ui5`）

| 指令      | 编码（[31:15]）       | 功能描述                                      |
|-----------|----------------------|-----------------------------------------------|
| `SLLI.W`  | `17'b00000000010000001` | `rd = rj << ui5`，逻辑左移（立即数）         |
| `SRLI.W`  | `17'b00000000010001001` | `rd = rj >> ui5`，逻辑右移（立即数）         |
| `SRAI.W`  | `17'b00000000010010001` | `rd = rj >>> ui5`，算术右移（立即数）         |

### 三、12 位立即数指令（6 条，格式：`op rd, rj, si12`）

| 指令      | 编码（[31:22]）  | 功能描述                                                  |
|-----------|-----------------|-----------------------------------------------------------|
| `ADDI.W`  | `10'b0000001010` | `rd = rj + SignExt(si12)`，立即数加法                    |
| `SLTI`    | `10'b0000001000` | `rd = (rj < SignExt(si12)) ? 1 : 0`，有符号立即数比较   |
| `SLTUI`   | `10'b0000001001` | `rd = (rj < SignExt(si12)) ? 1 : 0`，无符号立即数比较（立即数符号扩展后按无符号数比较）|
| `ANDI`    | `10'b0000001101` | `rd = rj & ZeroExt(si12)`，立即数按位与                  |
| `ORI`     | `10'b0000001110` | `rd = rj \| ZeroExt(si12)`，立即数按位或                 |
| `XORI`    | `10'b0000001111` | `rd = rj ^ ZeroExt(si12)`，立即数按位异或                |

### 四、20 位立即数指令（2 条）

| 指令          | 编码（[31:25]） | 格式               | 功能描述                                   |
|---------------|----------------|--------------------|--------------------------------------------|
| `LU12I.W`     | `7'b0001010`   | `op rd, si20`      | `rd = si20 << 12`，加载高 20 位立即数      |
| `PCADDU12I`   | `7'b0001110`   | `op rd, si20`      | `rd = PC + (si20 << 12)`，PC 加立即数偏移 |

### 五、跳转指令（5 条）

| 指令    | 编码（[31:26]） | 格式                    | 功能描述                                                      |
|---------|----------------|-------------------------|---------------------------------------------------------------|
| `BEQ`   | `6'b010110`    | `beq rj, rd, offs16`    | 若 `rj == rd`，则 `PC = PC + SignExt(offs16 << 2)`           |
| `BNE`   | `6'b010111`    | `bne rj, rd, offs16`    | 若 `rj != rd`，则 `PC = PC + SignExt(offs16 << 2)`           |
| `B`     | `6'b010100`    | `b offs26`              | 无条件跳转：`PC = PC + SignExt(offs26 << 2)`                 |
| `BL`    | `6'b010101`    | `bl offs26`             | 调用：`r1 = PC + 4`，`PC = PC + SignExt(offs26 << 2)`        |
| `JIRL`  | `6'b010011`    | `jirl rd, rj, offs16`   | 间接跳转：`rd = PC + 4`，`PC = rj + SignExt(offs16 << 2)`   |

### 六、访存指令（8 条，格式：`op rd, rj, si12`，地址 = `rj + SignExt(si12)`）

| 指令     | 编码（[31:22]）   | 功能描述                                          |
|----------|------------------|---------------------------------------------------|
| `LD.W`   | `10'b0010100010`  | 从内存加载 32 位字，写入 `rd`                     |
| `LD.B`   | `10'b0010100000`  | 加载 1 字节，有符号扩展至 32 位，写入 `rd`       |
| `LD.BU`  | `10'b0010101000`  | 加载 1 字节，零扩展至 32 位，写入 `rd`           |
| `LD.H`   | `10'b0010100001`  | 加载 2 字节（半字），有符号扩展至 32 位，写入 `rd` |
| `LD.HU`  | `10'b0010101001`  | 加载 2 字节（半字），零扩展至 32 位，写入 `rd`   |
| `ST.W`   | `10'b0010100110`  | 将 `rd` 的 32 位值存入内存                        |
| `ST.B`   | `10'b0010100100`  | 将 `rd[7:0]` 存入内存对应字节                    |
| `ST.H`   | `10'b0010100101`  | 将 `rd[15:0]` 存入内存对应半字                   |

## CPU 架构说明

本 CPU 为**单周期**实现，每条指令在一个时钟周期内完成取指、译码、执行、访存、写回全部流程。

### 流水线阶段（单周期合并执行）

```
取指 (IF)  →  译码 (ID)  →  执行 (EX)  →  访存 (MEM)  →  写回 (WB)
```

### 关键设计点

| 模块         | 说明                                                                                   |
|-------------|----------------------------------------------------------------------------------------|
| **PC**       | 复位值为 `0x1C000000`（LoongArch 标准复位地址），支持顺序 +4 和跳转两种更新方式         |
| **寄存器堆** | 32 个 32 位寄存器，r0 硬连线为 0（读永远返回 0，写操作忽略），同步写、异步读           |
| **ALU**      | 12 种操作的独热编码控制，加减法共用加法器，有符号/无符号比较均由减法器结果推导         |
| **访存**     | 数据存储器 4 KB，字节写使能（4 位 `we`），支持字节/半字/字的非对齐字节位提取          |
| **跳转**     | `BEQ`/`BNE` 通过比较器判断；`B`/`BL` 使用 26 位偏移；`JIRL` 支持寄存器间接跳转       |

### 立即数扩展方式

| 立即数字段   | 扩展方式                     | 使用指令                                        |
|-------------|-----------------------------|-------------------------------------------------|
| `si12`      | 符号扩展到 32 位             | `ADDI.W`、`SLTI`、`SLTUI`、`LD.*`、`ST.*`     |
| `ui12`      | 零扩展到 32 位               | `ANDI`、`ORI`、`XORI`                           |
| `ui5`       | 零扩展到 32 位               | `SLLI.W`、`SRLI.W`、`SRAI.W`                   |
| `si20`      | 左移 12 位，低 12 位补 0     | `LU12I.W`、`PCADDU12I`                          |
| `offs16`    | 符号扩展后左移 2 位          | `BEQ`、`BNE`、`JIRL`                            |
| `offs26`    | 符号扩展后左移 2 位          | `B`、`BL`                                       |
